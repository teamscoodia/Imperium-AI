
## Project Overview
Title: "Choose Your Mentor" 
This project showcases "three mentor cards" — "Machiavelli", "Napoleon", and "Aurelius" 

Each card subtly fades at the edges, gently tilts on hover/tap, and glows with a golden rim effect, creating a smooth, immersive experience.

## Tech Stack
- Framework:Next.js
- Styling: Tailwind CSS  
- Animation: Framer Motion  
- Fonts: Cinzel (Headings), Inter (Body)  
- Hosting: Vercel  


## Key Features
1. Side-Fade Blending
- Each card uses a CSS gradient mask to create soft edge fading.  

2. Interactive Motion
- On hover/tap, each card gently tilts and gains a gold rim glow.  
- Animations are powered by Framer Motion and fine-tuned CSS transitions.

3. Empire Energy Bar
- A pulsing gold “V” chevron appears at the bottom center.  
- On click, it smoothly scrolls to a status strip below.  


## Responsiveness
- Designed mobile-first, adapts seamlessly to all screen sizes.  
- Uses flex/grid layouts 
- Tested across mobile and desktop browsers for consistency.
